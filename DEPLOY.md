# Como publicar no GitHub Pages

Passo a passo para subir o projeto e ativar o GitHub Pages.

---

## 1. Criar o repositório

1. Acesse [github.com/new](https://github.com/new)
2. **Repository name:** `ciclo-prestacao-contas`
3. Deixe **Public** marcado
4. **NÃO** marque "Add a README file"
5. Clique em **Create repository**

---

## 2. Enviar os arquivos

Na tela que aparecer após criar o repositório, clique em **"uploading an existing file"** e envie os 3 arquivos:

- `index.html`
- `README.md`
- `404.html`

Depois clique em **Commit changes**.

---

## 3. Ativar o GitHub Pages

1. No repositório, clique em **Settings** (engrenagem)
2. No menu lateral, clique em **Pages**
3. Em **Source**, selecione **Deploy from a branch**
4. Em **Branch**, selecione **main** e pasta **/ (root)**
5. Clique em **Save**

---

## 4. Acessar o sistema

Após 1-2 minutos, o sistema estará disponível em:

**https://eduardocarloscardoso.github.io/ciclo-prestacao-contas**

---

## Atualizar o sistema no futuro

Para publicar uma nova versão:
1. Acesse o repositório no GitHub
2. Clique em `index.html`
3. Clique no ícone de lápis (editar)
4. Substitua o conteúdo ou use o botão de upload
5. Clique em **Commit changes**

O site atualiza automaticamente em 1-2 minutos.
